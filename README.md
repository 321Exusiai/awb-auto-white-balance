# Darkroom Nonclassic - 智能图像预处理工具包

## 项目概述
本项目是一个基于 Python 开发的专业数字图像预处理工具包，旨在修复无人机航拍、水利场景等户外图像的色彩偏差、亮度失衡等问题，同时提供边缘检测可视化功能，辅助目标识别与场景分析。

项目采用模块化设计，集成了多种经典的自动白平衡算法、伽马校正算法和边缘检测算法，提供现代化的 GUI 界面，支持多图批量管理、处理历史记录回退等功能，兼具实用性与可扩展性。

## 核心功能
### 1. 自动白平衡（AWB）
- **基础版灰度世界算法**：基于“全图RGB均值相等”的假设，实现快速白平衡校正。
- **优化版灰度世界算法**：在基础版上加入中亮度筛选和高亮区域保护，避免蓝天、窗户等高亮区域偏色。
- **完美反射体算法**：基于“图像最亮区域为纯白”的假设，支持高光像素占比调节，适合有白色参照物的场景。
- **基础版灰度边缘算法**：基于 Sobel 边缘检测，仅对边缘区域做灰度世界校正，免疫大面积纯色干扰。
- **优化版灰度边缘算法**：在基础版上加入高亮区域保护，鲁棒性更强。

### 2. 伽马校正
- **手动伽马校正**：支持 0.1~3.0 的伽马值调节，γ>1 图像变亮（优化暗部），γ<1 图像变暗（优化过曝）。
- **自动伽马校正**：基于直方图自动计算最优伽马值，将图像平均亮度调整到 128（中等亮度），实现一键优化。

### 3. 边缘检测可视化
- 基于 Sobel 算子的边缘检测，支持阈值调节。
- 将检测到的边缘用醒目的红色叠加在原图上，半透明显示，既保留原图信息，又突出边缘区域。
- 可用于识别河道边界、水利设施、水体异常区域等，对接智慧水利与遥感航拍场景。

### 4. 多图管理与历史记录
- 支持批量导入多张图片，提供网格/列表两种视图模式。
- 每张图片独立保存处理状态、算法参数和历史记录。
- 支持历史记录回退，可随时恢复到之前的处理版本。
- 支持键盘左右键快速切换图片。

## 算法原理简述
### 自动白平衡
所有白平衡算法基于经典的对角成像模型：$I_c(x,y) = R_c(x,y) \cdot e_c$，其中 $I_c$ 是拍摄图像，$R_c$ 是物体真实反射率，$e_c$ 是光源增益。算法的核心是估算 $e_c$ 并还原 $R_c$。

### 伽马校正
基于幂函数变换：$O = 255 \times (I/255)^{1/\gamma}$，解决人眼非线性感知与物理设备线性记录的不匹配问题。

### 边缘检测
基于 Sobel 算子计算图像梯度，提取颜色/亮度变化的区域，公式为：$G = \sqrt{G_x^2 + G_y^2}$，其中 $G_x$ 和 $G_y$ 分别是 x 和 y 方向的梯度。

## 技术栈
- **编程语言**：Python 3.8+
- **GUI 框架**：CustomTkinter（现代化暗色主题）
- **图像处理**：OpenCV, NumPy
- **图像显示**：Pillow (PIL)

## 使用说明
### 1. 安装依赖
```bash
pip install -r requirements.txt
```
### 2. 运行项目
```bash
python awb_gui.py
```
### 3. 基本操作流程
- 启动项目后，点击「📁 导入图片」，选择一张或多张图片；
- 在图片库中点击选中要处理的图片，或双击查看大图；
- 点击「✏️ 编辑」切换到处理视图；
- 在侧边栏选择算法，调节参数（如有）；
- 点击「✨ 执行校正」处理图片，或直接拖动伽马滑块实时预览；
- 点击「💾 导出结果」保存处理后的图片；
- 处理完成后，可在历史记录面板中回退到之前的版本。
